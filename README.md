# 🐾 Ranger 🐾

### "Every tail tells a story. We’re making sure it’s a happy one."

**Ranger** is the flagship landing page and waitlist portal for **FetchMyHeart**, an intelligent adoption platform dedicated to reducing shelter returns. By moving away from "breed-first" searching and focusing on lifestyle-based machine learning compatibility, we ensure the first home is the only home.

---

## 🚀 The Mission
Studies show that up to 15% of adopted dogs are returned within six months. FetchMyHeart (Project Ranger) aims to solve "serial rehoming" by:
* **Compatibility Matching:** Using data to match dogs with families based on energy levels and home environments.
* **Stability Focus:** Reducing animal stress by ensuring successful first-time placements.
* **The Pack:** Inspired by the unique personalities of Daisy, Ziba, and Ranger.

## 🛠 Tech Stack
* **Framework:** Next.js 14 (App Router)
* **Styling:** Tailwind CSS
* **Icons:** Lucide-React
* **Lead Capture:** Formspree Integration
* **Design:** Custom "Journey" UI with fluid background trail
